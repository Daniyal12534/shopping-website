# 🛍️ DANIYALMART - Premium Shopping Website

A fully functional, professional shopping system built with pure HTML, CSS, and JavaScript. No frameworks, no installation — just open and use!

## 🌐 Live Demo
[Click here to visit DANIYALMART](https://daniyal12534.github.io/shopping-website/)

## ✨ Features

- 🛒 Add to cart with quantity controls
- 🔍 Live search bar
- 📦 Category filter (Electronics, Fashion, Home, Beauty, Sports, Food)
- 💳 Full checkout form with order confirmation
- 🏷️ Sale & New badges, star ratings
- 🚚 Free shipping over $100
- 🌙 Dark luxury theme (Gold & Black)
- 📱 Mobile responsive
- 🎉 Unique order ID on every purchase

## 🚀 How to Run

1. Download `index.html`
2. Double-click to open in any browser
3. Done!

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript
